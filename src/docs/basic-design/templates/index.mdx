# 基本設計 テンプレ使い方

このセクションでは、基本設計工程で作成する **主要な成果物のテンプレート** をまとめています。  
画面設計やデータベース設計、非機能設計など、後続工程で参照される重要な成果物が含まれます。

---

## 📌 この階層の位置づけ 
- **目的**: 基本設計成果物を標準化し、設計品質を均一化する  
- **内容**: 各テンプレートの構成、利用シーン、記載のコツ

- システム方式設計書

- データベース論理設計書
